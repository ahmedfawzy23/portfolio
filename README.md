## Installation

1. Clone the repository:
```bash
git clone https://github.com/ahmedfawzy23/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Open `index.html` in your web browser or use a local server:
```bash
# Using Python
python -m http.server

# Using Node.js
npx serve
```
